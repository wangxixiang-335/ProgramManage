# 学生端成果发布功能

## 功能概述
学生端成果发布功能允许学生：
1. 发布各种类型的学术成果
2. 保存草稿功能
3. 富文本编辑器编写内容
4. 上传封面图、演示视频和附件
5. 选择审批人进行成果审核
6. AI辅助润色和布局优化

## 支持的成果类型
- 其他
- 网站开发
- 数据分析
- 游戏开发
- 移动应用
- 办公应用
- 创意作品
- 人工智能

## 数据库字段映射
- `id`: UUID主键
- `title`: 成果标题
- `description`: 项目描述
- `type_id`: 成果类型ID (UUID，关联achievement_types表)
- `cover_url`: 封面图片URL
- `video_url`: 演示视频URL
- `status`: 审核状态 (draft/pending/approved/rejected)
- `score`: 评分
- `publisher_id`: 发布者ID (UUID)
- `instructor_id`: 指导教师ID (UUID)
- `parents_id`: 父级成果ID (UUID)
- `created_at`: 创建时间 (timestamptz)

## 使用方法
1. 访问 `/achievement-publish` 路由
2. 填写基本信息（标题、描述、类型）
3. 上传封面图
4. 添加合作伙伴和指导老师
5. 使用富文本编辑器编写详细内容
6. 上传演示视频和附件
7. 预览或直接发布

## 文件上传限制
- 封面图：JPG、PNG格式，建议尺寸1200×675
- 演示视频：MP4、MOV格式，时长不超过5分钟，大小不超过200MB
- 附件：最多5个，单个不超过50MB

## AI功能
- AI润色：智能优化文本内容
- AI布局：自动优化内容布局和结构

## 注意事项
- 必填字段：标题、成果类型、封面图、内容
- 发布前需要至少选择一位审批人
- 草稿可以随时编辑和发布
- 支持从草稿箱继续编辑
